# CS Native Brand

CS Native follows the eust-w public profile direction: technology for good, robotics systems, VLA/VLM workflows, simulation data, and observable local runtimes.

## Brand Idea

CS Native is a local control plane for Claude Science. The brand should feel like a precise robotics runtime rather than a generic chat or proxy tool.

Product-wise, CSN is a provider/runtime control plane. It should not be presented as a single DeepSeek/Qwen switcher; those are only built-in presets in a broader provider profile system.

## Visual System

- Base: dark graphite and midnight blue, matching engineering dashboards and runtime consoles.
- Accent colors: cyan for model/data routing, green for healthy local runtime, violet for VLM/agent intelligence, and amber for simulation/real-world feedback.
- Shape language: rounded macOS squircle, looped node graph, and compact status chips.
- Mark: use `CSN` as the app identifier. Include a small `eust-w` signature in the lower-left runtime panel area where the icon has enough room, while keeping the CSN mark readable at Dock and Finder sizes.
