## Quid Sit

CS Native non est simplex commutator DeepSeek/Qwen. Est tabula localis ad administrandum:

- Provider profiles: base URL, API key, adapter, exemplar primarium, et Science model mapping.
- Proxy localem, OAuth virtualem, arcam Claude Science, et fresh nonce public entry.
- Modum provider externi intra arcam separatam.
- Modum officialem Claude separatim, ad verum iter Claude Science.

## Usus Primarius

1. Aperi **Models & Keys**.
2. Elige provider preset inclusum vel crea provider proprium.
3. Imple `base URL`, `API key`, `adapter`, exemplar primarium, et Science model mapping.
4. Preme **Verify**.
5. Redi ad **Start**, elige provider activum cum key servata, et preme **Start**.
6. Muta ad **Official Claude** si verum iter Claude Science opus est.

## Fines Executionis

- Directorium configurationis: `~/.csnative`
- Sandbox HOME: `~/.csnative/sandbox/home`
- Acta localia: `~/.csnative/logs/csnative.log`
- Provider keys in `~/.csnative/config.json` cum permissione `0600` servantur.
- CS Native in verum directorium credentialium Claude Science non scribit.
